# BankHisobi
Bank Tizimi — C#, Windows Forms va SQL Server texnologiyalari asosida yaratilgan desktop dastur. Tizim mijozlar ma'lumotlarini boshqarish, omonatlarni yuritish, amallarni qayd etish va hisobotlarni shakllantirish imkoniyatlarini taqdim etadi. Dastur qulay interfeys va ma'lumotlarni xavfsiz saqlash xususiyatlariga ega.
