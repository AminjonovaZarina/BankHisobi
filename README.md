# 🏦 BankHisobi

BankHisobi — Python dasturlash tilida yaratilgan bank hisobini boshqarish tizimi. Ushbu dastur foydalanuvchilarga mijozlar ma'lumotlarini saqlash, hisoblarni boshqarish, mablag'larni kiritish va yechish, hamda hisobotlarni ko'rish imkonini beradi.

## ✨ Asosiy imkoniyatlar

* Mijoz qo'shish
* Mijoz ma'lumotlarini tahrirlash
* Mijozlarni o'chirish
* Hisob ochish
* Balansni tekshirish
* Pul kiritish
* Pul yechish
* Hisobotlarni ko'rish
* Ma'lumotlarni saqlash va yuklash

## 🛠️ Foydalanilgan texnologiyalar

* Python 3
* Tkinter (GUI)
* SQLite / JSON (agar ishlatilgan bo'lsa)
* Object Oriented Programming (OOP)

## 🚀 O'rnatish

Repository'ni klon qiling:

```bash
git clone https://github.com/AminjonovaZarina/BankHisobi.git
```

Loyiha papkasiga o'ting:

```bash
cd BankHisobi
```

Dasturni ishga tushiring:

```bash
python main.py
```
